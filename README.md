CashOops is a full-stack expense tracking application that helps users record, organize, and visualize their daily and monthly expenditures.  
The project uses FastAPI for the backend and React (Vite) for the frontend.

---

## Features

- User authentication (email + password)
- Add, edit, delete expenses
- Categorize expenses
- Monthly spending summaries
- Category-based summaries
- Dashboard with charts
- Secure JWT authentication
- Fully deployed frontend and backend

Planned features:

- Google login
- Budget alerts
- CSV export
- Dark mode
- Smart category suggestions

---

## Technology Stack

### Backend

- FastAPI
- SQLAlchemy
- PostgreSQL (Neon or Railway)
- Python 3.x

### Frontend

- React (Vite)
- React Router
- React Query
- Chart.js
- Axios
